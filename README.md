# Price Tracker

Сервис для мониторинга и анализа цен товаров, спроектирован для обработки данных от парсеров
Также поддерживает загрузку данных вручную на соответствующих эндпоинтах.

 ## Как запустить проект локально:

1. **База данных:**
   Создать PostgreSQL базу `price_tracker` и пользователя (параметры по умолчанию в `Database.scala`).
   
2. **Миграции:**
  При первом запуске Flyway автоматически создаст таблицу `prices` из папки `src/main/resources/db/migration`.

3. **Запуск сервера:**
   ```bash
   sbt run
   ```

## Ключевые особенности
* ### Batch Ingestion
  Поддержка массовой вставки данных (/batch_ingest). Использование updateMany в Doobie минимизирует количество транзакций и нагрузку на IO.
* ### Валидация данных
  Валидация входящих данных. Ошибочные записи логируются и отсеиваются через partitionMap, не прерывая обработку всей пачки.
* ### Resource Safety
  Управление подключениями к БД и серверу через Resource, закрытие пула соединений (HikariCP) при остановке приложения.
* ### История записей
  Система хранит полную историю изменений цен для аналитики. Пишется всё на уровне БД (TIMESTAMP)

## Эндпоинты API
* POST `api/prices/ingest` - отправка одиночного замера цены на товар. Передается id товара, название товара, название магазина, цена. ID записи и время загрузки в БД проставляется автоматически

Запрос:
  ```JSON
  {
        "productId": string,
        "productName": string,
        "storeName": string,
        "price": double
  }
  ```

* POST `api/prices/batch_ingest` - отправка набора замеров цен на различные товары

Запрос:
  ```JSON
  [
    {
        "productId": string,
        "productName": string,
        "storeName": string,
        "price": double
    },
    {
        "productId": string,
        "productName": string,
        "storeName": string,
        "price": double
    }
  ]
  ```

* GET `api/analytics/get_price_stats/{productId}` - получение средней, максимальной и минимальной цены товара по `productId`

Ответ:
```JSON
{
    "average": double,
    "max": double,
    "min": double
}
```
